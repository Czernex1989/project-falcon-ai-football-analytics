# AI Football Analyst Knowledge Map

This document describes the main knowledge areas required to develop toward an AI Football Analyst role.

The goal is to combine football understanding, data analysis, SQL, Python, AI-assisted workflows and clear reporting.

## 1. Football understanding

Football analysis starts with understanding the game.

Key areas:

- formations
- player roles
- attacking phase
- defensive phase
- transitions
- pressing
- counter-pressing
- set pieces
- space creation
- overloads
- team structure
- movement without the ball

The analyst should not only describe what happened, but also understand why it happened.

## 2. Match analysis

Match analysis means looking for patterns, not only watching the result.

Important questions:

- How does the team build attacks?
- Where does the team lose the ball?
- How does the team defend after losing possession?
- Which zones are used most often?
- Which players are responsible for progression?
- What tactical patterns repeat during the match?
- How does the team react after substitutions?
- What weaknesses can be exploited by the opponent?

The goal is to move from opinion to structured observation.

## 3. Football data

Football data helps support tactical conclusions with evidence.

Important concepts:

- xG
- xA
- shots
- key passes
- progressive passes
- final third entries
- possession
- PPDA
- duels
- recoveries
- interceptions
- heatmaps
- shot maps
- event data
- tracking data

Data should support analysis, not replace football understanding.

## 4. SQL and Python

SQL is useful for asking questions from structured football data.

Example SQL questions:

- Which players created the most chances?
- Which team had the most shots?
- Which players completed the most progressive passes?
- How many shots came from inside the box?
- Which team lost possession most often in dangerous zones?

Python is useful for:

- cleaning data
- analyzing CSV files
- creating charts
- working with football datasets
- building notebooks
- automating reports
- preparing portfolio projects

SQL helps ask clear questions. Python helps analyze and visualize the answers.

## 5. AI-assisted workflow

AI can support the analyst by speeding up repetitive and structural work.

AI can help with:

- organizing observations
- summarizing notes
- creating report structure
- comparing data
- finding possible patterns
- drafting first versions of reports
- generating analysis questions
- improving documentation
- explaining technical concepts

AI does not replace football judgment.

The analyst is still responsible for checking the logic, context and final conclusion.

Project motto:

**AI przyspiesza pracę, a człowiek nadaje jej sens.**

## 6. Reporting

A good football analytics report should be clear, structured and useful.

A report should explain:

- what happened
- why it happened
- whether it was a pattern or a one-time situation
- what data supports the conclusion
- what recommendation follows from the analysis

Possible report types:

- post-match team report
- opponent analysis
- player report
- scouting report
- tactical report
- set-piece report
- pressing analysis
- data-based player comparison

The final goal is not only to collect information, but to turn it into useful football conclusions.

## 7. Portfolio projects

First Project FALCON football analytics portfolio ideas:

- basic match report
- simple player comparison
- xG explanation report
- team style analysis
- SQL football mini-project
- Python football data chart
- AI-assisted match analysis workflow
- football metrics glossary
- match analysis template

Each project should show visible progress and practical learning.

## 8. Learning direction

The learning path should move step by step:

1. Understand football analysis basics.
2. Learn key football metrics.
3. Practice SQL with simple football-style data.
4. Practice Python with CSV files and charts.
5. Create simple analysis templates.
6. Build small public portfolio projects.
7. Combine AI, data and football understanding into clear reports.

## Summary

AI Football Analyst combines:

- football understanding,
- match analysis,
- football data,
- SQL,
- Python,
- AI-assisted work,
- clear reporting.

The goal of Project FALCON is to build this skill set gradually through public, documented and practical work.
