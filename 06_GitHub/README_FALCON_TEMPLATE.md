# Project FALCON

Project FALCON is a structured career-transition project focused on AI evaluation, SQL, Python and football analytics.

The goal is to turn daily learning into visible work, visible work into portfolio proof, and portfolio proof into better career opportunities.

## Main Tracks

- AI Evaluation and Search Quality
- SQL for Data Analysis
- Python Foundations
- Football Analytics
- GitHub and Portfolio Development

## Current Focus

I am currently building practical foundations in SQL and Python while developing a football analytics portfolio. The project is based on a 60-day execution plan with daily work, small projects and regular documentation.

## Repository Structure

- `01_sql` - SQL exercises, notes and mini-projects
- `02_python` - Python exercises, scripts and notebooks
- `03_football_analytics` - match reports, player reports and football data projects
- `04_ai_evaluation` - general notes on AI evaluation workflows and learning progress
- `05_portfolio` - summaries of completed projects
- `06_docs` - roadmap documents and templates
- `data` - sample data only
